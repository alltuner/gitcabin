# Code of Conduct

This project adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

In short: be kind, be patient, assume good faith, and disagree on the technical points without making it personal. Treat the comment threads here the way you'd treat a conversation in a small workshop with people you respect.

## Reporting

If you see behavior that violates the Contributor Covenant, contact the maintainers privately at **david@poblador.com**. Reports will be handled confidentially.

For routine technical disagreement, use the issue tracker — that's not a Code of Conduct concern, that's just open-source.
